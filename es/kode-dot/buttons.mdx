# Características

EL kode dot dispone de un pad direcinal de 4 botones más dos botones independientes. Mediante estos botones se puede navegar por el kodeOS, controlar el estado del ESP32-S3 y apagar/encender el dispositivo.

## Esquema de conexión
Todos los botones tienen una resistencia de pull-up y están conectados al expansor de pines I2C, exceptuando el botón de arriba que está conectado directamente al ESP32-S3.

<Info>Ve a TBD para más información sobre el expansor de pines.</Info>


El botón de arriba está conectado al pin GPIO0, para controlar el BOOT del ESP32-S3. Los demás botones están conectados al expansor de pines de esta manera:

| Botón         | Expansor de pines |
| ------------  | ----------------- |
| Pad Izquierdo | P7                |
| Pad Arriba    | P6                |
| Pad Derecha   | P13               |
| Pad Abajo     | P10               |
| Botón Abajo   | P11               |
