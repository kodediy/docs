# Características

El corazón del kode dot es el microprocesador ESP32-S3 de Espressif. Tiene una flash de 32MB y una PSRAM de 8MB.

## Control de estados: BOOT y RESET
El control de los estados de BOOT y RESET se realiza mediante la combinación de los botones del kode dot.

Para resetear el ESP32-S3, se debe pulsar el botón izquierdo del pad a la vez que se pulsa el botón de abajo.

Para entrar en modo BOOT, mientras se mantiene el botón de arriba, se debe resetear el kode dot pulsando el botón izquiedo del pad a la vez que se pulsa el botón de abajo.
