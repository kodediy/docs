# Comunidad de kode.
<AccordionGroup>
  <Accordion title="¿Qué es kode?">
    kode es una comunidad de código abierto para aprender, construir y crear tus ideas.
  </Accordion>

  <Accordion title="¿Dónde puedo entrar a la comunidad?">
    Puedes entrar a la comunidad de kode en [Discord](https://discord.gg/kode).
  </Accordion>
</AccordionGroup>

# kode dot
<AccordionGroup>
  <Accordion title="¿Qué es kode dot?">
    kode dot es un dispositivo de hardware que te permite crear tus ideas.
  </Accordion>

  <Accordion title="¿Dónde puedo comprar el kode dot?">
  Puedes comprar el kode dot en [kode.diy](https://kode.diy).
  </Accordion>

  <Accordion title="¿Cómo puedo programar el kode dot?">
    Puedes programar el kode dot con el firmware de kodeOS.
  </Accordion>
</AccordionGroup>

# kodeOS
<AccordionGroup>
  <Accordion title="¿Qué es kodeOS?">
    kodeOS es el sistema operativo del kode dot.
  </Accordion>
  
  <Accordion title="¿El kode dot viene ya con el kodeOS?">
  El kode dot viene ya con el kodeOS.
  </Accordion>

  <Accordion title="Si borro sin querer el kodeOS, ¿cómo puedo recuperarlo?">
    Puedes recuperar el kodeOS con el firmware de kodeOS.
  </Accordion>
</AccordionGroup>
